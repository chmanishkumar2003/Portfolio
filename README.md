# Portfolio Website

This is my personal portfolio website showcasing my skills, projects, and experiences.  
The portfolio also contains my **resume**, which can be viewed and downloaded directly from the site.  

## Features
- Modern and responsive design  
- Sections: Home,About Me, Education, Projects, Skills, Contact  
- Integrated Resume (view and download option)  
- Built with HTML, CSS, and JavaScript (can be extended with frameworks like React/Next.js)  

## Purpose
The portfolio serves as an online professional profile to highlight my journey as a developer, share my work, and provide easy access to my resume for recruiters and collaborators.  

## How to Use
1. Clone the repository  
2. Open `portfolio.html` in your browser  
3. Navigate through sections to explore resume and projects
